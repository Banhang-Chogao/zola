Điểm mấu chốt nếu làm theme này

  • Grid dày, editorial: hero 3 cột + khối chuyên mục lặp lại là điểm khác lớn nhất so với theme blog hiện tại (single column + sidebar). Cần template index.html mới.
  • Màu nhấn magenta (#e11d63) — znews dùng rất tiết chế: thanh mục, gạch đầu dòng, tab active. Nếu áp cần thêm token --c-accent biến thể (không đè token brand hiện tại theo rule light/dark).
  • Trang chuyên mục khác hẳn: tiêu đề lớn canh giữa + tab con + lead/rail. Cần sửa section.html / taxonomy_single.html.
  • Ảnh là trung tâm: mọi thẻ đều có thumbnail. Blog cần đảm bảo mỗi bài có cover (đã có owned-cover fallback) — hợp lệ.
  • Responsive: ≤820px hero 3 cột → 1 cột, carousel 6→2, nav thu về menu ☰ (theo rule R1–R8, chỉ sửa trong media query mobile).
  • Chưa động chạm gì: file này nằm ở wireframes/ (ngoài content/ & static/) nên KHÔNG lên site production, chỉ để bạn xem & quyết định.
