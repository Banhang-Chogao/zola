### рҹ“қ Mini CMS viбәҝt bГ i trб»ұc tiбәҝp trГӘn trГ¬nh duyб»Үt

### рҹҺҜ Featured & Random Posts cГЎ nhГўn hoГЎ
Sidebar JS Д‘б»Қc `posts-data` JSON inline, **weighted random** dб»ұa trГӘn view / click count lЖ°u `localStorage` вҖ” mб»—i visitor thбәҘy mб»ҷt danh sГЎch khГЎc nhau.

### рҹҡҰ GitHub Live Status Banner
Header rotator fetch GitHub API real-time Д‘б»ғ hiб»ғn thб»Ӣ:
- Commit message + hash + timestamp mб»ӣi nhбәҘt
- Pending workflow queue (GitHub Actions Д‘ang chбәЎy)

Comment system dГ№ng **GitHub Discussions** lГ m backend вҖ” khГҙng spam, khГҙng cбә§n auth riГӘng, mб»Қi comment Д‘б»Ғu lГ  issue/discussion thбәӯt trГӘn repo.

### Autofixer (merge conflict PR)

Workflow `autofix-conflicts.yml` chбәЎy mб»—i 30 phГәt (vГ  `workflow_dispatch`):

1. QuГ©t PR open bб»Ӣ conflict vб»ӣi `main`
2. TбәЎo nhГЎnh `autofix/conflict-pr-<N>`, merge `main`, resolve markers an toГ n
3. ChбәЎy `qa_check.py` вҶ’ `build_references.py` вҶ’ `zola build` вҶ’ `check_internal_links.py`
4. Mб»ҹ PR fix riГӘng вҖ” **khГҙng** push `main`, **khГҙng** auto-merge

State dedup: `data/autofix-conflicts-state.json`. Learning log: `CLAUDE.md` В§ Autofixer.

### Adding a New Theme (Guide for Developers)

#### Step 1: Create Token Files

```bash
# Create color & typography tokens
touch sass/_mytheme-tokens.scss
touch sass/_mytheme-fonts.scss
```

**Example: `_mytheme-tokens.scss`**
```scss
$mytheme-primary:      #COLOR;
$mytheme-accent:       #COLOR;
$mytheme-border:       #COLOR;
$mytheme-radius-sm:    4px;
$mytheme-shadow-sm:    0 2px 4px rgba(0,0,0,0.08);
```

#### Step 2: Define Theme Variables in `_themes.scss`

```scss
:root[data-theme="mytheme"] {
  --c-bg-page:        #fff;
  --c-text-heading:   #000;
  --c-accent:         #COLOR;
  --c-border:         #COLOR;
  --c-shadow-md:      /* shadow value */;
  /* ...add all 15+ CSS variables... */
  font-family: 'Your Font', fallback, sans-serif;
}
```

#### Step 3: Add Component Overrides in `_theme-overrides.scss`

```scss
:root[data-theme="mytheme"] {
  @include theme-overrides(
    $radius-card:        4px,
    $radius-tag:         4px,
    $shadow-card:        $mytheme-shadow-sm,
    $shadow-card-hover:  $mytheme-shadow-lg,
    $kicker-spacing:     0.05em,
    $heading-ls:         -0.02em
  );
}
```

#### Step 4: Register in JavaScript

**`static/js/theme-switcher.js`**
```javascript
var VALID_THEMES = ["hilda"];  // Single theme: Hilda (Ericsson Blue)
```

#### Step 5: Update Imports

**`sass/site.scss`**
```scss
@import "mytheme-tokens";
@import "mytheme-fonts";  // if custom font
```

#### Step 6: Update Toggle Button

**`templates/base.html`** (optional, if supporting 3+ themes)
```html
<!-- Dropdown or multi-choice instead of binary toggle -->
```
