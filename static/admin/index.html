⚙️ CMS chỉ hoạt động tại máy local

Bạn đang truy cập URL production (…). Decap CMS không có backend trên GitHub Pages.

Để viết bài bằng CMS, làm tại máy của bạn:

  1. Clone repo: git clone https://github.com/Banhang-Chogao/zola
  2. cd zola
  3. Mở 2 terminal, chạy song song:
    • Terminal 1: npx decap-server (port 8081)
    • Terminal 2: zola serve (port 1111)
  4. Truy cập http://localhost:1111/admin/
  5. Viết bài, click Publish — file .md tự ghi vào content/
  6. Khi xong: git add + git commit + git push
