ĐĂNG NHẬP CMS

Cần GitHub Personal Access Token để viết bài

Token sẽ được lưu trong trình duyệt này (localStorage) để CMS có thể đẩy file .md trực tiếp vào repo. Không gửi đi đâu khác.

Cách tạo token (1 lần, ~2 phút)
  1. Mở GitHub → Settings → Developer settings → Personal access tokens → New token (classic)
  2. Note: Blog CMS
  3. Expiration: 90 days (hoặc No expiration nếu chấp nhận rủi ro)
  4. Scopes: tick repo (full control of repositories)
  5. Click Generate token → copy chuỗi ghp_xxxxxxxx...
  6. Dán vào ô dưới đây
