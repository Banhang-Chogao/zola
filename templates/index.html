Static site engine tất-cả-trong-một của bạn

Quên các dependency đi. Mọi thứ bạn cần đều nằm trong một file binary duy nhất.

Bắt đầu ngay

Không phụ thuộc

Zola là một file thực thi duy nhất, đã tích hợp sẵn biên dịch Sass, syntax highlighting, mục lục và rất nhiều tính năng khác mà bình thường bạn phải cài đặt thêm môi trường hoặc thêm các thư viện JavaScript vào trang web của mình.

Nhanh như chớp

Một trang web trung bình sẽ được build trong chưa đến một giây, bao gồm cả biên dịch Sass và syntax highlighting.

Khả năng mở rộng

Zola render toàn bộ trang web thành file tĩnh, giúp bạn xử lý mọi loại lưu lượng truy cập một cách dễ dàng mà không phải lo lắng về việc quản lý server hay cơ sở dữ liệu.

Dễ sử dụng

Từ giao diện dòng lệnh đến template engine, mọi thứ đều được thiết kế để trực quan và dễ dùng. Đừng tin lời tôi nói — hãy xem tài liệu và tự kiểm chứng.

Linh hoạt

Zola không cản đường bạn, để bạn tập trung vào nội dung — dù là blog, trang kiến thức, landing page, hay kết hợp tất cả lại.

Markdown nâng cao

Zola tích hợp sẵn shortcode và liên kết nội bộ giúp việc viết nội dung của bạn trở nên dễ dàng hơn.
