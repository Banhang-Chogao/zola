{# ============================================================================ Macro ảnh WebP — phục vụ bản .webp (nhẹ hơn) kèm fallback file gốc. Dùng kèm pipeline scripts/to_webp.py + workflow optimize-images.yml: với mỗi ảnh nội bộ .jpg/.png đã có sẵn bản .webp cùng tên cạnh bên. Macro này render để browser hỗ trợ webp lấy bản nhẹ, browser cũ fallback file gốc. CHỈ dùng cho ảnh NỘI BỘ (static/...). Ảnh ngoài (picsum, CDN) không có .webp nên cứ dùng thường. Tham số: - src: đường dẫn ảnh gốc (vd "/img/covers/abc.jpg") - alt: alt text - cls: class (optional) - loading: "lazy" (mặc định) | "eager" - sizes/width/height: optional, tốt cho CLS ============================================================================ #} {% macro picture_webp(src, alt="", cls="", loading="lazy", width="", height="") %} {%- set ext = src | split(pat=".") | last -%} {%- set webp = src | replace(from="." ~ ext, to=".webp") -%} {%- endmacro picture_webp %}
