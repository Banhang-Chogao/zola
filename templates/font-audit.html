Font Audit

Công cụ audit font giúp kiểm tra font thật đang render trên production. Khai báo font-family trong CSS chưa chắc là font browser đang dùng — cần computed style hoặc render thật qua Chromium.

Vì sao không có form nhập URL?

Blog deploy trên GitHub Pages (static-only). JavaScript phía client không thể đọc computed style của trang cross-origin vì same-origin/CORS. Audit URL bất kỳ cần browser automation — chạy qua CLI trên máy bạn hoặc CI.

Cách chạy

Cài Playwright (khuyến nghị) để lấy computed font thật:

pip install -r scripts/requirements-font-audit.txt
playwright install chromium

Audit một URL và lưu JSON cho trang này:

python3 scripts/font_audit.py "https://seomoney.org/baochi/bi-kip-xin-visa-han-quoc-5-nam-de-baochi/" \
  --json data/font-audit-latest.json

Nếu Playwright chưa cài, script vẫn chạy ở chế độ fallback (chỉ parse CSS/HTML) và đánh dấu computed font là unknown.
