{# ===== MANIFESTO ===== #}

Design Manifesto

Tối giản đến tận cùng. Hiện đại đến từng pixel.

BrandingX không phải một bộ rule. Đây là một cách suy nghĩ: mỗi quyết định thiết kế phục vụ rõ ràng một mục đích người dùng.

{# ===== 1. COLOR PALETTE ===== #}
01

Color Palette

Color system 3 tầng: Primary (brand), Secondary (support), Accent (alert + emphasis). Mỗi màu có 5 sắc độ neutral 50-900.

Primary — Brand

Ink 900 #0f172a Heading, primary text
Ink 800 #1e293b Body text
Ink 500 #64748b Muted text, captions

Accent — Highlight

Signal Red #d63d3d CTA, active state, error
Hot Pink #ec4899 Category tags, decorative
Trust Cyan #0891b2 Link, info, finance category

Surface — Background

Snow #ffffff Default surface
Mist #f8fafc Soft section bg
Edge #ececec Borders, dividers
Tư duy: Palette ưu tiên contrast cao (WCAG AAA cho text body). Pink + Red không pair cùng layer — pink dùng cho tags decoration, red cho action. Cyan dành riêng finance/info — tránh dùng cho action vì gây nhầm lẫn link.
{# ===== 2. TYPOGRAPHY ===== #}
02

Typography

Single typeface Manrope cho mọi role. Geometric round nhưng vẫn cleaner so với competitors. Hỗ trợ Tiếng Việt diacritics hoàn hảo.

Display
3rem (48px) / 800 / -0.02em / lh 1.1 Hero, landing headlines
Heading 1
2rem (32px) / 800 / -0.02em / lh 1.2 Page title, hero card
Heading 2
1.5rem (24px) / 700 / lh 1.25 Section title
Heading 3
1.15rem (18.4px) / 700 / lh 1.3 Card title, subsection
Body text — phù hợp cho đoạn dài 1.65 line-height vừa thoáng vừa đọc nhanh.
1rem (16px) / 400 / lh 1.65 Paragraph body
Caption — meta info, date, author byline.
0.85rem (13.6px) / 500 / muted Metadata, captions
Tư duy: Single typeface giảm cognitive load + load 1 web font request. Manrope round friendly nhưng bold 800 vẫn có authority cho heading. Letter-spacing âm cho heading lớn (>24px) để chữ "đậm" thị giác hơn, không nhỏ cho body.
{# ===== 3. UI COMPONENTS ===== #}
03

UI Components

4 atom cơ bản: Button, Card, Input, Icon. Mọi composition phức tạp xây từ 4 atom này.

Buttons

3 tier: Primary (1 per view), Secondary (alternative), Ghost (tertiary/cancel).

Card

Card title

Subtle border, no shadow at rest. Hover lifts +2px.

Flat at rest, depth on interaction. radius 12px.

Input

Stacked label, single border, focus ring 3px accent color.

Icon

○ △ □ ◇

Stroke 2px, square 24×24 viewbox. Geometric primitives ưu tiên.

{# ===== 4. GRID & SPACING ===== #}
04

Grid & Spacing

12-column grid, max-width 1280px. Spacing scale 8px base. Mobile-first responsive.

TokenValueUse
--space-xs4pxIcon gap, tight inline
--space-sm8pxCard body gap
--space-md16pxStandard rhythm
--space-lg24pxSection internal
--space-xl32pxSection gap
--space-2xl48pxMajor section divide
Tư duy: 8px base = chia chẵn cho mọi viewport, đồng bộ với Material/Apple HIG. 12-cột chia hết cho 2/3/4/6 — flexible cho mọi layout. Max-width 1280px = optimal reading + còn margin sides cho 1440p+ monitors.
