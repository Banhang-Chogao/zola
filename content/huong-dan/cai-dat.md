Zola được phân phối dưới dạng một file binary duy nhất, không có dependency. Tải về và bỏ vào thư mục `PATH` là dùng được ngay.

## Cài trên Linux

```bash
curl -sLO https://github.com/getzola/zola/releases/latest/download/zola-v0.20.0-x86_64-unknown-linux-gnu.tar.gz
tar xzf zola-*.tar.gz
sudo mv zola /usr/local/bin/
```

## Cài trên macOS

```bash
brew install zola
```
