Đây là bài viết đầu tiên của blog. Trang web này được tạo bằng [Zola](https://www.getzola.org/),
một static site engine cực kỳ nhanh, gói gọn trong một file binary duy nhất.

<!-- more -->

## Vì sao chọn Zola?

- **Một file binary** — không cần Node, không cần Ruby, không cần Python. Tải về dùng ngay.
- **Tốc độ build** — site trung bình build dưới 1 giây.
- **Tích hợp sẵn** — Sass, syntax highlighting, search index, mục lục, taxonomy, RSS.
- **Markdown nâng cao** — shortcodes, internal links, footnotes.

Hãy đọc thêm các bài viết khác để biết cách dùng Zola hiệu quả.
